This repository contains simple JavaScript programs created to understand two important JavaScript concepts: **Hoisting** and **Scope**.

The goal of this practice is not just to write code, but to understand what happens behind the scenes when JavaScript executes a program.


## Topics Covered

### Hoisting
- var Hoisting
- let Hoisting
- const Hoisting
- Function Declaration Hoisting
- Function Expression Hoisting
- Temporal Dead Zone (TDZ)

### Scope
- Global Scope
- Function Scope
- Block Scope
- Scope Chain
- Variable Shadowing
- Difference between var, let, and const

 Programs Included

### Program 1 – Hoisting Practice

This program demonstrates:
- How `var` behaves when accessed before declaration
- Why `let` and `const` throw errors before initialization
- How function declarations can be called before they are defined
- Understanding JavaScript's memory creation and execution phases

### Program 2 – Scope Checker

This program demonstrates:
- Global Scope
- Function Scope
- Block Scope
- Accessibility of variables in different scopes
- Real examples showing where variables can and cannot be accessed


## What I Learned

Through this practice, I learned:

- How JavaScript hoists variables and functions.
- The difference between `var`, `let`, and `const`.
- What the Temporal Dead Zone (TDZ) is.
- How JavaScript manages variable visibility using scope.
- The difference between Global, Function, and Block Scope.
